My Research

Thesis: Predicting E. coli Drug Resistance Using Traditional Machine Learning and Deep Learning

My research centers on building computational pipelines and machine learning models that can predict antibiotic resistance in E. coli. For my thesis, I focus on a single antibiotic Ciprofloxacin(CIP) and use a two complementary datasets: gene presence/absence and single-nucleotide polymorphisms (SNPs). The goal is to understand how specific features influence resistance to CIP and develop predictive models that can identify resistant strains based on their genomics.

The Problem

Antibiotic-resistant infections pose a serious global health threat. One of the main causes is drug resistant (E. coli). Usually, testing if a bacterial strain is resistant takes a lot of time in the lab. But, now with faster genome sequencing and machine learning, we can try to predict resistance directly from DNA sequences. The challenge is that an E.coli genome has thousands of genes and mutations, and not all of them are important for resistance. Finding the right signals in all that data is hard task, and that's where machine learning can help.

My Solution & Methods

I tackled this challenge by developing a machine learning pipeline that turns raw genomic data into actionable predictions of antibiotic resistance. My approach leverages complementary genomic feature sets and state-of-the-art ML techniques:

  • Data: I used whole-genome sequences from 588 E. coli strains, each labeled as Resistant or Susceptible to Ciprofloxacine.
  • Feature Engineering: I extracted two types of genomic features to capture different scales of variation:
    • Gene Presence/Absence Matrix: This dataset contains 6,093 genes and shows whether each gene is present or absent in every strain. These genes mainly come from the shell genome, which includes genes shared by some, but not all strains. Constant features were removed.
    • SNP Variant Profiles: This dataset includes 166,415 single-nucleotide polymorphisms (SNPs) across all strains. SNPs were filtered using minor allele frequency (MAF) and gap threshold to remove low-quality SNPs.
  • Model Training: With these feature sets, I trained and tuned classification models to predict resistance. I experimented with various machine learning algorithms (e.g. logistic regression,random forests, Random Forest, SVM, etc.) and used rigorous cross-validation to avoid overfitting the high-dimensional data. Handling thousands of genomic features required careful feature selection and regularization to focus the models on the most informative signals.
  • Evaluation & Comparison: I evaluated model performance separately using the gene-presence features and the SNP features to compare their predictive power. This analysis revealed which type of genetic signal is more indicative of resistance. I also explored combining both feature types in one model to capture both large-scale and fine-scale genetic factors, aiming for improved overall accuracy and robustness.
