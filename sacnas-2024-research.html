Pangenome Pipeline for Predicting Antibiotic Resistance in E. coli

Presented at SACNAS National Conference ‚Ä¢ October 2024

San Francisco State University ‚Ä¢ CoDE Lab

Problem

Previous studies on antibiotic resistance prediction in E. coli typically focus on either SNPs data or gene presence/absence patterns. This approach limits model generalizability across different laboratories and datasets. We developed a pangenome framework that integrates both SNPs and gene presence/absence data to predict resistance to multiple antibiotics in bacterial populations.

Solution

We built a pangenome framework that integrates multiple datasets and combines SNPs with gene presence/absence features. We used various bioinformatics tools to create the pangenome framework and extract features from the data. The goal of this poster is to demonstrate our pangenome pipeline framework and its potential for antibiotic resistance prediction.

Pipeline Highlights

  • Integrated datasets from Mills (~2000), Gladstone (~3000), and Kalloneen (~2000) samples.
  • Constructed a pangenome framework across multiple datasets.
  • Feature space: SNPs + gene presence/absence.
  • Models tested: Previous studies used logistic regression, random forest, gradient boosting. We plan to use the entire pangenome framework to train models and compare performance with previous approaches.

Results

  • Established foundation for comprehensive ML model comparisons
  • Presented findings at SACNAS 2024 National Conference
